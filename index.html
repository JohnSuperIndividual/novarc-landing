Quickstart

One line to switch from OpenAI

NovArc is 100% OpenAI-compatible. Change one URL and your existing code works instantly — no other modifications needed.

- base_url="https://api.openai.com/v1"   # OpenAI: ~$15/M output tokens
+ base_url="https://new-api-production-d6db.up.railway.app/v1"  # NovArc: ~$1.25/M output tokens

from openai import OpenAI

client = OpenAI(
    base_url="https://new-api-production-d6db.up.railway.app/v1",
    api_key="your-novarc-key",
)

response = client.chat.completions.create(
    model="deepseek-chat",
    messages=[{"role": "user", "content": "Hello!"}],
)
print(response.choices[0].message.content)

💡 That's it. Same SDK, same format — 90% cheaper.
